Matrix Chain Multiplication (矩陣鏈乘積)

設定與輸入
Rows Cols
請輸入第一個矩陣的維度。
矩陣 維度 (Rows x Cols) 操作
遞迴關係式 (Recursive Formula)
$$ s[i, j] = \begin{cases} 0 & \text{if } i = j \\ \min_{i \le k < j} \{ s[i, k] + s[k+1, j] + p_{i-1}p_k p_j \} & \text{if } i < j \end{cases} $$
其中 $s[i, j]$ 為計算 $A_i \dots A_j$ 的最小純量乘法次數。
速度: 1500 ms
準備就緒
最佳括號方式: 尚未計算
圖示說明： 左側表格 s[i][j] 顯示最小運算量，右側表格 c[i][j] 顯示最佳切割點 k。
請輸入矩陣並點擊執行。
